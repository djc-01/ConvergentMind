# GitHub Discussion Starters

Use these drafts after enabling GitHub Discussions for the repository.

## 1. Project introduction and feedback thread

Title: `Welcome to ConvergentMind: feedback, first impressions, and use cases`

Body:

```md
ConvergentMind is a small, local-first research platform for browser-first multimodal agents.

It combines:

- browser perception
- optional camera or video-stream context
- explicit observe -> plan -> act -> verify loops
- replayable local artifacts for debugging

If you try the repo, I would love to hear:

1. what workflow you tested
2. where the current loop felt brittle
3. what extension point you would most want next

Links:

- README: https://github.com/Chocologician/ConvergentMind
- Roadmap: https://github.com/Chocologician/ConvergentMind/blob/main/ROADMAP.md
```
