## Draft qualification summary

ConvergentMind is an actively maintained open source research platform for browser-first multimodal agents. It gives developers a clear observe-plan-act-verify loop that combines browser perception, optional camera context, local replay artifacts, and OpenAI-compatible planning interfaces. I maintain the docs, tests, releases, issue triage, and contributor workflow so the project stays usable as a small but extensible foundation for agent experimentation and teaching.

## Maintainer responsibilities

- triage issues and discussions
- maintain README, examples, roadmap, and release notes
- keep tests and packaging healthy
- review and merge contributions
- manage GitHub releases and PyPI publishing
