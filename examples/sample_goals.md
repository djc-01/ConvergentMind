# Sample Goals

These are example task prompts you can use while experimenting with ConvergentMind.

## Safe local demo goals

- `Fill the form and submit it.`
- `Inspect the page, then submit the form after filling both fields.`
- `Use the camera frame as extra context, but finish the browser task safely.`

## Browser-agent research goals

- `Describe the visible controls before taking any action.`
- `Extract the main page text, then explain which interactive element is most likely to advance the task.`
- `Click the primary button only after verifying that the page state matches the goal.`

## Future multimodal goals

- `Use the camera stream to detect whether a person is present, then decide whether to continue the browser task.`
- `Compare what the browser asks for with what is visible in the external camera scene.`
- `Wait until the external scene changes, then continue the workflow in the browser.`

Use these as prompt ideas, not as guarantees that the current implementation already supports the full behavior.
