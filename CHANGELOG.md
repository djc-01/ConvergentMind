# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows Semantic Versioning as a practical release convention.

## [0.1.0] - 2026-06-01

### Added

- Initial public release of the ConvergentMind local-first research platform.
- Browser observation and browser control loop with replayable run artifacts.
- Optional camera or video-stream snapshots as a second perception channel.
- OpenAI-compatible planner and verifier integrations plus an offline demo mode.
- CLI commands for `run`, `inspect`, `replay`, and `doctor`.
- Core contributor docs, release guidance, CI workflow, and GitHub collaboration templates.
